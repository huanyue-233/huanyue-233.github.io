    • 多重背包问题(MultiplePack)

    问题模型

    有一个容量为WWW的背包和nnn种物品,第iii个物品有pip_ipi​件,重量为wiw_iwi​,价值为viv_ivi​,最大化总价值

    解析

    相似于完全背包,我们可以枚举kkk表示选取的数量

    fi,j=max(fi−1,j−wi×k+k×vi)∣0≤k×wi≤min(pi,⌊W/wi⌋)\Large f_{i,j}=max(f_{i-1,j-w_i \times k}+k \times v_i)| 0\leq k \times w_i \leq min(p_i,\lfloor W/w_i \rfloor)fi,j​=max(fi−1,j−wi​×k​+k×vi​)∣0≤k×wi​≤min(pi​,⌊W/wi​⌋)

    优化

    依然相似于完全背包,可以通过拆分物品和二进制优化到O(W×∑i=1nlog2p[i])O(W \times \sum \limits _{i=1}^n log_2p[i])O(W×i=1∑n​log2​p[i])基本够用

    for (int i = 1; i <= n; i++) {
    int num = min(p[i], W / w[i]);

                while (head < tail && que[head].pos < k - p[i])

                f[k * w[i] + mo] = que[head].value + k * v[i];
