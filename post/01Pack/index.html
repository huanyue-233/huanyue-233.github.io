    问题模板

    有一个背包总容量为WWW,有nnn个物品,第iii个物品的重量为wiw_iwi​,价值为jij_iji​

    最大化价值

    解析

    对于每个物品都有取与不取的两种状态,如果暴力搜索,复杂度为2n2^n2n

    如何使用动态规划来解决

    状态设计

    fi,jf_{i,j}fi,j​表示只使用前iii个物品,容量为jjj的背包的最大价值

    转移方程

    考虑已经处理好了i−1i-1i−1的情况,那么第iii个物品,取或者不取

    那么有fi,j=max(fi−1,j,fi−1,j−wi+vi)f_{i,j}=max(f_{i-1,j},f_{i-1,j-w_i}+v_i)fi,j​=max(fi−1,j​,fi−1,j−wi​​+vi​)

    for (int i = 1; i <= n; i++)
    for (int j = W; j >= w[i]; j--)
        f[j] = max(f[j], f[j - w[i]] + v[i]);

    优化

    显然的,用二维数组空间复杂度较大,用滚动数组优化

    因为对于fif_ifi​只由fi−1f_i-1fi​−1推出,那么滚掉第一维,可以直接得出

    dj=max(dj,di−wj+vj)d_j=max(d_j,d_{i-w_j}+v_j)dj​=max(dj​,di−wj​​+vj​)*这里djd_jdj​的定义容量为iii的背包的最大价值

    我们要保证在计算djd_jdj​时,djd_jdj​与dj−wid_{j-w_i}dj−wi​​与fi−1,j,fi−1,j−wif_{i-1,j},f_{i-1,j-w_i}fi−1,j​,fi−1,j−wi​​

    由于j−wi≤jj-w_i\leq jj−wi​≤j,那么只要从大到小更新djd_jdj​,就不会影响后面的状态

    因此从WWW到000枚举jjj即可

    代码

    #include <iostream>
using namespace std;
const int N = 13010;
int n, W, w[N], v[N], f[N];
int main() {
  cin >> n >> W;
  for (int i = 1; i <= n; i++) 
      cin >> w[i] >> v[i];
  for (int i = 1; i <= n; i++)
    for (int l = W; l >= w[i]; l--)
      if (f[l - w[i]] + v[i] > f[l]) f[l] = f[l - w[i]] + v[i];
  cout << f[W];
  return 0;
}
