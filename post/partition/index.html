    • 分治法(partition)

    概述

    很多算法都是递归的: 为了解决一个给定的问题,算法一次或多次递归地调用自身来解决紧密相关的若干个子问题.

    这些算法符合分治法的思想,分治,即"分而治之":意思是分别治理；利用手段使国家、民族或宗教等产生分裂，然后对其进行控制和统治。出自《群经平议·周官二》 " 凡邦之有疾病者，疕疡者造焉，则使医分而治之，是亦不自医也。"

    分支算法在每层递归的时候都有三个步骤:

    分解

    将原问题分为若干个子问题,这些子问题是原问题的规模较小的实例

    解决

    解决这些子问题,递归地求解各个子问题,如果子问题的规模足够小,则直接求解

    合并

    合并这些子问题的解成原问题的解.

    实例:归并排序(merge_Sort)

    归并排序是O(nlogn)O(n log n)O(nlogn)的排序算法中的一种,稳定并且严格O(nlogn)O(n logn)O(nlogn).

    程序流程

    将给定的包含nnn个元素的数组分割为两个局部数组,每个数组个包含n/2n/2n/2个元素

    对两个局部数组分别执行mergeSortmergeSortmergeSort排序.

    通过mergemergemerge将两个已排序完毕的局部数组"整合"成一个数组.

    在归并排序中,合并两个已排序的数组的mergemergemerge是整个算法的基础,要将包含n1n1n1个整数的数组LLL以及包含n2n2n2个整数的数组RRR合并道数组AAA.如果我们在合并时直接套用普通的排序算法,显然就浪费了其各自已经排序的性质.接下来介绍mergemergemerge的实现

    merge

    由于两个数组都已经按照优先值排列,因此,在前面的元素输出前,后面的元素一定不会被输出.因此,用两个指针分别指向LLL，RRR的首个未输出元素，只要输出两个元素中的优先元素即可．为了方便,我们可以在两个数组后放置两个哨兵.

    复杂度分析

    因为我们通过二分递归,会将数据分为log2nlog_2nlog2​n层,每层mergemergemerge的复杂度为O(n)O(n)O(n),那么总复杂度就实现了O(nlogn)O(nlogn)O(nlogn)

    ####　代码示例

    #include <iostream>
using namespace std;
const int N = 500000, inf = 0x7fffffff;
int L[N / 2 + 2], R[N / 2 + 2];

void merge(int A[], int left, int mid, int right) //需要排序的数组,起点,分割点,终点+1
{
    int n1 = mid - left;
    int n2 = right - mid;
    for (int i = 0; i < n1; i++)
        L[i] = A[left + i];
    for (int i = 0; i < n2; i++)
        R[i] = A[mid + i];
    L[n1] = R[n2] = 0x7fffffff; //设置哨兵
    int i = 0, j = 0;           //指针
    for (int k = left; k < right; k++)
        if (L[i] <= R[j]) //带等号,维持稳定的性质
            A[k] = L[i++];
        else
            A[k] = R[j++];
}

void mergeSort(int A[], int left, int right)
{
    if (left + 1 < right)
    {
        int mid = ((left + right) >> 1);
        mergeSort(A, left, mid);
        mergeSort(A, mid, right);
        merge(A, left, mid, right);
    }
}

int main()
{
    int A[N], n; //当然可以将数组开在全局,省去递归时数组的调用
    cin >> n;
    for (int i = 1; i <= n; i++)
        cin >> A[i];
    mergeSort(A, 1, n + 1);
    for (int i = 1; i < n; i++)
    {
        cout << A[i] << " ";
    }
    cout << A[n] << endl;
    return 0;
}
