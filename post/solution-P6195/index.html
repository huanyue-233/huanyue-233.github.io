    证明

    假设我们现有的数最多能表示出[1,k][1,k][1,k]内所有的整数

    增加一个数,使得之后得k′k'k′最大.

    设这个数为ttt

    ∵ [1,k][1,k][1,k]之间的整数都可以表示出来

    ∴ t+1,t+2,t+3.....t+k−1,t+jt+1,t+2,t+3.....t+k-1,t+jt+1,t+2,t+3.....t+k−1,t+j即[t+1,t+k+1][t+1,t+k+1][t+1,t+k+1]间得整数都可以表示

    ∵要令区间连续

    ∴t+1≤k+1t+1\leq k+1t+1≤k+1

    t≤kt\leq kt≤k

    又∵要令k′k'k′最大

    ∴要令t+k+1t+k+1t+k+1最大

    t=kt=kt=k时t+k+1t+k+1t+k+1最大,此时能表现出得区间为[1,2k+1][1,2k+1][1,2k+1]

    结论: 每次新增一个数可以让原有数表示出的区间大小×2+1\times 2 +1×2+1

    那么问题就简化成这样了:

    f(x)=f(x−1)×2+1,f(1)=nf(x)=f(x-1)\times2+1,f(1)=nf(x)=f(x−1)×2+1,f(1)=n

    求f(m)f(m)f(m)

    考后的我(这种证明颇为有趣,但也有一定知识要求):

    巧妙的利用二进制

    ×2+1\times 2+1×2+1相当于二进制中的<<1∣1<<1|1<<1∣1

    例如(D 表示 十进制 B 表示二进制):

    10D=1010B10D=1010B10D=1010B

    10D×2+1=21D=10101B10D \times 2 +1 =21D =10101B10D×2+1=21D=10101B

    简单的来说,就是在二进制码的最后面塞一个111

    那么我们要将n×2+1 mn\times 2 +1\ mn×2+1 m次,相当于在二进制码后面塞了mmm个111

    这mmm个111的和是20+21+22+...+2m−2+2m−1=2m−12^0+2^1+2^2+...+2^{m-2}+2^{m-1}=2^{m}-120+21+22+...+2m−2+2m−1=2m−1

    我们nnn左移了mmm位,也就是×2m\times 2^m×2m

    那么结果为n×2m+2m−1=(n+1)×2m−1n \times 2^m+2^m-1=(n+1)\times 2^m-1n×2m+2m−1=(n+1)×2m−1

    #include <iostream>
#include <cstdio>

typedef long long ll;
using namespace std;
const int mod = 1000000007;

inline int read()
{
    int x = 0, f = 1;
    char ch = getchar();
    while (ch < '0' || ch > '9')
    {
        if (ch == '-')
            f = -1;
        ch = getchar();
    }
    while (ch >= '0' && ch <= '9')
    {
        x = ((x << 3) + (x << 1) + (ch ^ 48));
        ch = getchar();
    }
    return x * f;
}
ll qpow(int b)
{
    ll ans = 1, res = 2;
    while (b)
    {
        if (b & 1)
            ans = ans * res % mod;
        res = res * res % mod;
        b >>= 1;
    }
    return ans % mod;
}

int main()
{
    cout << ((read() + 1) * qpow(read()) - 1) % mod;
    return 0;
}
